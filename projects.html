Project Euler Solutions

My collection of solutions to the Project Euler problem set. Solutions are fully browsable in the project's GitHub repository.

Most of the website is auto generated using the GitHub API to grab file contents, and generate code previews.

Spotify Instant

Instant search for the popular music streaming application Spotify. It was developed using the Spotify API. This project was very well recieved, and reached the front page of Hacker News.

Try it here!

UW Open Data

A suite of RESTful APIs for UW services. It currently supports the University of Washington Directory and Course Catalog, which was developed by Karan Goel.

This API was developed in Python using Flask framework and BeautifulSoup. It was deployed using Heroku.

Flappy Leap Bird

A new derivation of the webapp implementation of Flappy Bird created by Nebez, which allows the player to control their bird using the Leap Motion Controller. This application was developed using the LeapJS API.

Try the demo here!

Leap Pong

A multiplayer version of the classic Pong for the Leap Motion.

I developed this application with Grant Timmerman, using the Leap Motion JavaScript SDK, Node.js and Socket.io for latency-free multiplayer capabilities.

NodeBlog

A full web stack blogging platform built on Node.js.

This platform uses a host of JavaScript frameworks, MongoDB and the Facebook JavaScript SDK to implement a fully socially integrated blogging platform, ready for immediate use.
